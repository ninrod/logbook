* passwordless sudo
** Ubuntu 22
tip from [[https://askubuntu.com/a/443071][here]]
   #+BEGIN_SRC emacs-lisp
     $ sudo visudo
     # in vim, change this
     #   %admin  ALL=(ALL) ALL
     # to this
     #   %admin  ALL=(ALL) NOPASSWD:ALL     

     # lastly, add your user to the admin group
     $ sudo usermod -a -G admin ninrod
   #+END_SRC
